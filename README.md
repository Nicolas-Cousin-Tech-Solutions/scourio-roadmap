# Scourio — Roadmap publique
 
Suivi public de l'avancement du développement de [Scourio](https://scourio.com), une plateforme de protection de marque et de surveillance SEO (détection d'empoisonnement SEO, de backlinks toxiques et de clones de site, avec signalement automatisé).
 
## Ce que contient ce repo
 
- Les milestones et issues de suivi du développement
- Le statut d'avancement par module
- Les retours des beta testeurs (anonymisés)
## Ce que ce repo ne contient pas
 
Le code source de Scourio est privé. Ce repo sert uniquement de suivi — pas de logique produit, pas de clés, pas de données sensibles.
 
## Statut actuel
 
🚧 En développement — Module 1 (surveillance SEO) en cours de construction.
